### Below you will find some information on how to perform common tasks.

To run the project on your dev machine:

```javascript
yarn start
```

To run tests:

```javascript
yarn test
```

To see test coverage:

```javascript
yarn run coverage
```

To run flow static type checking:

```javascript
yarn run flow
```

To build project for production:

```javascript
yarn run build
```
